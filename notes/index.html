      Sync across your devices

      Your notes are saved on this device automatically. To see the same notes on your phone, connect a private GitHub repository below. This is free and only you can see it.

      First time? Click here for step-by-step setup
      1. Open github.com/new and create a repository. Name it my-notes-data, choose Private, click Create repository.
      2. Open this token page. Set Token name = notes app, Expiration = No expiration (or 1 year), Repository access = Only select repositories → pick my-notes-data.
      3. Under Permissions → Repository permissions, find Contents and set it to Read and write. Click Generate token and copy the long text it shows.
      4. Paste the repository name and that token into the boxes below and press Save & connect.
