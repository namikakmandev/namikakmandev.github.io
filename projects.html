Operations · Automation · Data engineering · HTML · JS · Python

From scattered files to one report — in 28 seconds

Most weekly reports get rebuilt by hand from data scattered across a shared folder, an email attachment, the bank portal and a WhatsApp thread. This pulls it all together, cleans and cross-checks it, and assembles a single diagnostic report — surfacing what usually slips by: excess stock, late-paying customers, margin quietly leaking. Roughly 3 hours of weekly work, done in about 28 seconds. Built for fun, on a weekend.
